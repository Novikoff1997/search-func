Установка скрипта: 

1. Каждая карточка должна внутри себя содержать дополнительный "служебный" html тег который нужен
для того что бы содержать информацию для поиска. Это может быть любой элемент span, div и т.д.
Важно что бы этот элемент имел следующие атрибуты и конкретными названиями: 

    class='data-attributes' - По этому классу будет осуществляться получение этого элемента
    data-name="Анна Каренина" - Дада атрибут с названием книги
    data-category="classic" - Дата атрибут с названием категории к которой относится книга
    data-publisher="reading-city" - Дата атрибут с названием издательства
    data-year="1873" - Дата атрибут в годом 

2. Код в самом начале содержит переменные которым присваивается получение некоторых элементов со страницы
для управления поиском. К каждой такой переменной прописан комментарий о том что должно храниться в этой 
переменной. Поиск элементов осуществляется по классам, классы нужно заменить на те что используются в верстке.

    Пояснение:
        * Там где написано в скобках (родительский блок) - нужно указать класс блока родителя который содержит
        внутри себя элементы. Например если это категории, то нужен класс именно блока в котором
        лежат категории. Так аналогично для всех. 

        * Там где прописано что нужен конкретный элемент, указать класс этого элемента.

3. Элементы поиска и фильтрации должны содержать дата атрибуты значение которых равно тому значению которое
они будут искать. Например если это категория 'Классика' она должна сожержать дата атрибут со значением 
'classic' (или любой другой на ваше усмотрение) и все карточки подходящие под эту категорию должны иметь
у себя в дата атрибуте data-category точно такой же значение. 

В свою очередь элементы издательств должны иметь атрибут value значение которого имеет то же название
издательства что и соответствующий дата атрибут в карточке.

Какие атрибуты должны содержать элементы поиска: 

    data-filter="ваше значение" - применяется ко всем элементам категорий, содержит название категории
    value="ваше значение" - должно быть у элементов издательств, содержит название издательства (если чекбоксы)
    В случае если элементы издательств выполнены не через чекбоксы, то нужно написать мне об этом и я перепишу
    код под другие элементы.
